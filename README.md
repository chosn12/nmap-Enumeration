# nmap-Enumeration 

## 1. Install nmap:
   - Open Terminal
     
     ***`brew install nmap`***

## 2. Full Scan:
   - In Terminal

     ***`nmap -sS -SV -O -p- <target ip> -oX <file name.xml>`***

     This will run a full scan of the ip address and save the results in a file in XML format
